C programming

General-purpose, procedural language Developed by Dennis Ritchie in the 1970s at Bell Labs Known for efficiency and low-level memory access Ideal for system-level programming (e.g., operating systems, embedded systems) Highly portable due to standardized libraries and compilers

Java

General-purpose, object-oriented language Developed by James Gosling and released by Sun Microsystems in 1995 Known for its platform independence due to the Java Virtual Machine (JVM) Ideal for building cross-platform applications

Python

General-purpose, high-level language Developed by Guido van Rossum and first released in 1991 Known for its simplicity and readability Emphasizes code readability with its use of significant whitespace Extensive standard library and large ecosystem of third-party packages
